# PHOTO APP
See demo

This photo app was built in Rails 6.1.0, Ruby 2.6.3

GEMS USED:
 * Devise

 To run on your local machine:
 * Fork Repo
 * cd into it
 * Run `bundle install` to install the gems
 * Run `rails s` to start the server
